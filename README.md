# AWS Lambda Functions Manager

Use this package as AWS Lambda Functions Manager.<br />

### Install globally

```
npm i exec-lambda -g
lambda
```

### Install locally

```
npm i exec-lambda --save
cd exec-lambda
npm start
```

### Usage
The package show some options to manage your AWS lambda function and related layers

```
[1] Set layers for selected function
[2] Update layer to last version for selected function
[3] Update layer to last version to all related functions
[4] Download selected function
[5] Download all functions
```
